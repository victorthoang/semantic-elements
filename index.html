Task 2: Write one paragraph summarizing what semantic elements are (max of 150 words)

    Semantic elements are used as a basic building block for HTML. It is a common way to describe the layout and function of a webpage to both the end user and the browser.

  • For example: uses for a semantic tag is to describe the layout of a page. Common ones popularized by the HTML5 WG include: <header>, <footer>, <nav>, and <section> tags that describe the location description of the content embedded within it!
  • A header element can be used to describe an author at the top of the page, logo, icons, or even date edited.
  • A footer element can be used to describe copyright information, links back to the top, related documents/pages, or contact information.
  • There are many other semantic elements that can be used and found here
